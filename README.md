# Django/DRF based Task manager (W.I.P)

### Create venv

```bash
python -m venv .venv
```

### Activate venv

###### ⠀⠀Windows

```bash
.\.venv\Scripts\activate.ps1
```

###### ⠀⠀Linux

```bash
source .venv/bin/activate
```

### Install requirements

```bash
pip install -r requirements/requirements.txt
```

### Migrate database

```bash
python manage.py migrate
```

### Run server

```bash
python manage.py runserver
```
